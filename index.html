HTML

  • HTML stands for HyperTextMarkupLanguage
  • HTML acts as the structure of a webpage defining its content
  • Nested elements have a parent or can be a child of another element
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

Git

  • git is a tool that facilitates software version control
  • git is primarily used in the Command Line Interface (CLI) but there are apps and websites that have GUIS
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all files in a folder to the next commit
  • git commit -m "texthere": commits staged files to the relevant branch with comment "texthere"
  • git push: sends the committed files to github to be approved
