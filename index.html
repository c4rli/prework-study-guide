HTML

  • HTML stands for HyperText Markup Language
  • HTML acts as the structure of a webpage defining its content
  • Nested elements have a parent or can be a child of another element
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS can be inline i.e. within the HTML tag
  • CSS can also be an external file with .css extension

Git

  • git is a tool that facilitates software version control
  • git is primarily used in the Command Line Interface (CLI) but there are apps and websites that have GUIs
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all files in a folder to the next commit
  • git commit -m "texthere": commits staged files to the relevant branch with comment "texthere"
  • git push: sends the committed files to github to be approved

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript executes from top to bottom so defining elements before calling them is important.
  • Arrays are useful for grouping information, they can be of different data types.
